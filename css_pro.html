CSS Professional

Question 1

How can you create a custom scrollbar using CSS?

Answer

Use `::-webkit-scrollbar`, `::-webkit-scrollbar-thumb`, and other vendor-specific pseudo-elements.

Question 2

What is the difference between `fr` units and percentages in CSS Grid?

Answer

`fr` allocates available space proportionally; percentages are based on parent size regardless of space distribution.

Question 3

Explain how `contain` and `content-visibility` can improve performance.

Answer

`contain` isolates layout/rendering; `content-visibility: auto` skips rendering off-screen content.

Question 4

How do CSS custom properties differ from preprocessor variables (e.g., in Sass)?

Answer

CSS variables (`--var`) are dynamic and can be changed at runtime; Sass variables are static at compile time.

Question 5

What’s the purpose of `@supports` in CSS?

Answer

Used for feature detection: applies styles only if a browser supports a specific property/value.

Question 6

How do logical properties improve CSS layout for internationalization?

Answer

They abstract directions (e.g., `margin-inline-start`) to support LTR/RTL writing modes seamlessly.

Question 7

How does `display: contents` affect the DOM and layout?

Answer

Removes the box of the element while keeping its children rendered; children act as if they are direct descendants of the parent.

Question 8

Explain the difference between `auto-fill` and `auto-fit` in CSS Grid.

Answer

`auto-fill` maintains empty columns; `auto-fit` collapses them when no content is present.

Question 9

What’s the difference between `em` and `ex` units?

Answer

`em` is based on font size; `ex` is based on the x-height (height of lowercase letters).

Question 10

What are scroll snapping points and how are they implemented?

Answer

Implemented with `scroll-snap-type` on the container and `scroll-snap-align` on children for precise scroll positioning.

Question 11

What does the `accent-color` property control?

Answer

It sets the color of UI controls like checkboxes, radio buttons, and range sliders.

Question 12

What is the `:is()` pseudo-class used for?

Answer

Simplifies writing long selector groups: `:is(h1, h2, h3)` applies the same style to multiple elements.

Question 13

How does `object-fit` differ from `background-size`?

Answer

`object-fit` is used for replacing elements like ; `background-size` applies to background images.

Question 14

How can you make a CSS animation run once and stay at the end frame?

Answer

Use `animation-fill-mode: forwards;` with `animation-iteration-count: 1;`.

Question 15

What is the difference between `overflow: hidden` and `clip`?

Answer

`hidden` prevents scrollbars and cuts off overflow; `clip` is used in `overflow-clip-margin` or `clip-path` for precise clipping.

Question 16

How does the `:has()` pseudo-class enhance interactivity?

Answer

It selects a parent based on child conditions, e.g., `div:has(img)` targets divs that contain images.

Question 17

What is `forced-color-adjust` used for?

Answer

Overrides system color schemes in forced color modes (e.g., Windows High Contrast).

Question 18

How can you visually hide an element but keep it accessible?

Answer

Use `position: absolute; width: 1px; height: 1px; overflow: hidden; clip: rect(0, 0, 0, 0);`

Question 19

What does `mix-blend-mode` do?

Answer

Determines how an element’s content blends with background content.

Question 20

What is the purpose of `overscroll-behavior`?

Answer

Prevents scroll chaining and bounce effects when scrolling reaches the edge.

Question 21

What does `clip-path` do?

Answer

Defines a clipping region that determines what part of an element is visible.

Question 22

How can `grid-template-areas` improve readability?

Answer

Allows you to name and visually layout grid areas in a way that resembles the final layout structure.

Question 23

What is the default stacking context in CSS?

Answer

Each root element and elements with `position` other than `static` and a `z-index` create stacking contexts.

Question 24

How can you detect browser support for a CSS property?

Answer

Use `@supports (property: value)` feature queries.

Question 25

What’s the purpose of `appearance: none;`?

Answer

Removes native styling from form controls for full customization.

Question 26

What does `backdrop-filter` do?

Answer

Applies effects (like blur) to the area behind an element.

Question 27

What’s the difference between `min-content` and `max-content` in Grid or Flex?

Answer

`min-content` is the smallest size without overflow; `max-content` is the size needed for full content display.

Question 28

What’s the effect of `isolation: isolate` in a complex layout?

Answer

Prevents blending with parent/child layers by creating a new stacking context.

Question 29

How does `writing-mode` affect layout direction?

Answer

Changes text and layout flow direction to support vertical or horizontal writing systems.

Question 30

What is the benefit of `line-clamp` in modern CSS?

Answer

Allows truncation of multiline text with ellipsis using only CSS.

Question 31

How does `position: sticky` behave differently from `fixed`?

Answer

`sticky` toggles between relative and fixed based on scroll position; `fixed` stays pinned to the viewport.

Question 32

How does `aspect-ratio` simplify responsive design?

Answer

Preserves element proportions without needing padding hacks or JavaScript.

Question 33

What is a CSS Houdini API?

Answer

Low-level APIs that let developers extend CSS engine behavior (e.g., custom paint, layout).

Question 34

What are inert elements in CSS/HTML?

Answer

Elements with `inert` attribute are excluded from keyboard/tab navigation and interaction.

Question 35

How do container queries differ from media queries?

Answer

Container queries apply styles based on the size of a container, not the viewport.

Question 36

What is `scroll-margin` used for?

Answer

Adds space before an element when it is brought into view via scroll.

Question 37

Why might `will-change` hurt performance?

Answer

Overuse forces the browser to allocate memory and optimize unnecessarily.

Question 38

What is the difference between `initial`, `inherit`, and `unset`?

Answer

`initial`: default browser style, `inherit`: inherits from parent, `unset`: smart combo of both.

Question 39

How can CSS variables be scoped?

Answer

They inherit from their nearest ancestor with the variable defined, just like other inherited styles.

Question 40

What is the difference between `::before` and `:before`?

Answer

`::before` is the correct modern syntax; `:before` is legacy but still works in most browsers.

Question 41

How can you target dark mode users using CSS?

Answer

Use the media query `@media (prefers-color-scheme: dark) {}`.
