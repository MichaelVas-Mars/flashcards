CSS Intermediate

Question 1

What is the difference between `em` and `rem` units in CSS?

Answer

`em` is relative to the parent’s font size, `rem` is relative to the root (html) font size.

Question 2

How do you make a responsive layout using Flexbox?

Answer

Use `display: flex` and apply `flex-wrap: wrap` and media queries to control behavior on different screen sizes.

Question 3

What does the `calc()` function do in CSS?

Answer

It allows mathematical expressions to be used as property values, e.g., `width: calc(100% - 50px);`

Question 4

How do you center a div vertically and horizontally using Flexbox?

Answer

Use `display: flex; justify-content: center; align-items: center;` on the parent container.

Question 5

What’s the difference between `relative` and `absolute` positioning?

Answer

`relative` positions relative to its normal position; `absolute` positions relative to the nearest positioned ancestor.

Question 6

What does `minmax()` do in CSS Grid?

Answer

Defines a size range for grid tracks, e.g., `minmax(100px, 1fr)`.

Question 7

How do you apply a style only if a user hovers over an element’s parent?

Answer

Use a descendant selector like `.parent:hover .child { /* styles */ }`

Question 8

What does `object-fit: cover` do for an image?

Answer

Scales the image to fill the container while maintaining aspect ratio and cropping overflow.

Question 9

What is the specificity hierarchy in CSS?

Answer

Inline styles > IDs > Classes/pseudo-classes > Elements/pseudo-elements

Question 10

How do you create a responsive image gallery using CSS Grid?

Answer

Use `display: grid`, `grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));` and gap settings.

Question 11

What is the use of the `nth-child()` pseudo-class?

Answer

Selects elements based on their position in a parent: `li:nth-child(2)` selects the second child.

Question 12

How do you prevent an element from being affected by floating elements?

Answer

Use `clear: both;` or `overflow: auto;` on the container.

Question 13

What does `visibility: hidden` do?

Answer

Makes the element invisible but it still takes up space in the layout.

Question 14

How do you set a maximum width for an element?

Answer

Use the `max-width` property.

Question 15

What does `position: sticky` do?

Answer

Switches between relative and fixed depending on scroll position.

Question 16

How do you make a grid item span multiple rows?

Answer

Use `grid-row: start / end;` or `grid-row: span n;`

Question 17

What is the default value of the `z-index` property?

Answer

auto

Question 18

How do you create a CSS variable?

Answer

Define with `--name: value;` inside a selector. Use with `var(--name)`.

Question 19

What does `overflow: scroll` do?

Answer

Always shows scrollbars, even if not needed.

Question 20

What is the effect of `white-space: nowrap`?

Answer

Prevents text from wrapping to the next line.

Question 21

How can you animate an element using CSS?

Answer

Use `@keyframes` and the `animation` property.

Question 22

What’s the difference between `inline` and `inline-block`?

Answer

`inline-block` allows setting height and width; `inline` does not.

Question 23

How do you make a sticky footer using Flexbox?

Answer

Use `flex: 1` on content and set container to `display: flex; flex-direction: column; min-height: 100vh;`

Question 24

What’s the difference between `auto` and `0` margin?

Answer

`auto` can center elements; `0` sets no space.

Question 25

What does `box-sizing: border-box` do?

Answer

Includes padding and border in the total width/height.

Question 26

How do you apply styles only to the last child in a list?

Answer

Use `li:last-child` selector.

Question 27

What does `clip-path` do?

Answer

Clips an element to a specified shape.

Question 28

What’s the purpose of `pointer-events: none`?

Answer

Disables mouse interactions on the element.

Question 29

How can you apply multiple background images?

Answer

Separate images with commas in `background-image`.

Question 30

What is the difference between `:nth-child()` and `:nth-of-type()`?

Answer

`:nth-child()` counts all children; `:nth-of-type()` counts specific element types.

Question 31

What is the use of `backface-visibility` in CSS?

Answer

Controls whether the back face of an element is visible when rotated.

Question 32

What does `isolation: isolate;` do?

Answer

Creates a new stacking context to prevent blending with other elements.

Question 33

How do you apply a grayscale filter to an image in CSS?

Answer

Use `filter: grayscale(100%);`

Question 34

What does `scroll-behavior: smooth;` do?

Answer

Makes scrolling transitions smooth for anchor links.

Question 35

How do you target the first letter of a paragraph?

Answer

Use `p::first-letter` selector.

Question 36

What does `will-change` property do?

Answer

Hints to the browser what properties will change, for performance optimization.

Question 37

How do you apply styles to an element that has focus?

Answer

Use the `:focus` pseudo-class.

Question 38

What is the purpose of `@media (prefers-color-scheme: dark)`?

Answer

Applies styles based on the user's OS light/dark mode preference.

Question 39

What is the difference between `absolute` and `fixed` positioning?

Answer

`absolute` positions relative to the nearest positioned ancestor; `fixed` positions relative to the viewport.

Question 40

How do you make a flex item not grow in a flex container?

Answer

Use `flex-grow: 0;` on the item.

Question 41

What does the `gap` property do in Flexbox and Grid layouts?

Answer

Sets the spacing between rows and columns inside a flex or grid container.

Question 52

How do you select a sibling element that comes immediately after another?

Answer

Use the adjacent sibling selector: `element1 + element2`

Question 53

How do you create a triangle shape using CSS?

Answer

Use `border` on a zero-sized element with one colored side and the rest transparent.

Question 44

What property lets you change the stacking order of elements?

Answer

`z-index`

Question 45

What does `currentColor` refer to in CSS?

Answer

It refers to the value of the current `color` property of the element.

Question 56

What is the `all` shorthand property used for?

Answer

Resets all inheritable and non-inheritable properties to a specific value like `initial` or `unset`.

Question 47

How do you create an element that maintains a specific aspect ratio?

Answer

Use `aspect-ratio` or a padding-top percentage trick for older browser support.

Question 48

How do you apply a style only when printing the page?

Answer

Use a print media query: `@media print { ... }`

Question 49

What does `transition-timing-function: ease-in-out;` do?

Answer

Starts the transition slowly, speeds up in the middle, then slows down again.

Question 50

What is the purpose of `content: ""` in CSS?

Answer

Used with `::before` and `::after` pseudo-elements to generate content in the DOM.
